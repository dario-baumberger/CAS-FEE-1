# CAS-FEE-1
Todo App build with NodeJS.

## Requirements
- NodeJS
- Only modern Browsers are supported (ES6 and CSS Vars)

## Setup

`npm -i`

## Start

`npm run start`

[http://localhost:3000](http://localhost:3000)

## Features
- Add Notes
- Remove Notes
- Notes with due date
- Notes with states
- Notes with priority/importance
- Notes with categories
- Filter Notes
- Order Notes
- Light and Dark Mode
